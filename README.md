# Wordatro Helper 🎮

**Trouveur de mots longs pour Wordatro** — basé sur le dictionnaire officiel ODS (Scrabble francophone).

🔗 **[Ouvrir l'outil](https://[ton-username].github.io/wordatro-helper/)**

---

## Fonctionnalités

- 🔍 **Recherche instantanée** parmi 168 000+ mots (4 à 9 lettres)
- 🃏 **Support des jokers** (lettres wildcard)
- 📏 **Filtres par longueur** (4, 5, 6, 7, 8 ou 9 lettres)
- 🎯 **Filtre "commence par"** et **"contient"**
- 📊 **Tri** par longueur, score ou ordre alphabétique
- ⭐ **Score Scrabble** affiché pour chaque mot
- 💡 **Vue détaillée** par mot (quelles lettres utilisées, bonus longueur)

## Dictionnaire

Basé sur l'**ODS8** (Officiel du Scrabble, édition 8 — 2020), la référence officielle du Scrabble francophone utilisée par Wordatro.

411 430 mots au total · 168 447 mots de 4 à 9 lettres indexés.

## Déploiement GitHub Pages

1. Fork ce repo
2. Va dans **Settings → Pages**
3. Source : **Deploy from a branch** → `main` → `/ (root)`
4. Ton site sera disponible sur `https://[ton-username].github.io/wordatro-helper/`

## Structure

## Utilisation locale

Ouvre simplement `index.html` dans ton navigateur. Aucune installation requise, aucune dépendance externe (sauf Google Fonts).
